什么是Markdown

Markdown[1] 是一种方便记忆、书写的纯文本标记语言，用户可以使用这些标记符号以最小的输入代价生成极富表现力的文档：譬如您正在阅读的这份文档。它使用简单的符号标记不同的标题，分割不同的段落，粗体 或者 斜体 某些文字。Markdown文件的后缀通常为.md。

Markdown在流畅的书写和印刷级的阅读体验之间找到了平衡。目前它已经成为世界上最大的技术分享网站 GitHub 和 技术问答网站 StackOverFlow 的御用书写格式。

优点

  • 写作中添加简单符号即完成排版，所见即所得。让你专注于文字而不是排版。
  • 格式转换方便，Markdown的文本你可以轻松转换为HTML、PDF等。
  • 可以保存为纯文本。

几款好用的MarkDown编辑器

可以结合这篇文章选择合适的编辑器

  • Sublime Text 3
    • 本地
    • 需安装Markdown插件
    • 可快速生成HTML
    • 若要支持实时预览需更多插件且效果不佳
  • Typora
    • 本地
    • 无需插件
    • 可导出为带超链接的PDF，导出的html样式没有Sublime Text 3好
    • 所见即所得
  • Cmd Markdown
    • 国人出品，不开源
    • 在线编辑，分屏实时预览
    • 样式简约精美
    • 导出为PDF等高级功能需要收费
  • StackEdit
    • 国外出品，开源，Cmd Markdown的源头
    • 在线编辑，分屏实时预览
    • 样式简约精美
    • 导出为PDF等高级功能需要收费
